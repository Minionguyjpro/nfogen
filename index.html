The NFO generator project is aimed at creating a script that generates NFO files fast and conviniently for users without the need to do a lot of typing. Currently it has only one template for the NFO files, but in the future I have planned to add more.

The script is written in php and html and consists of a web page with a form on it with just a few spaces for the user to fill in. When the users submits the form a NFO file is generated for the user to see. If the results satisfies the users needs the NFo can be downloaded by the link provided in the top of the page. If not you can always go back and regenerate the nfo.
